# Politique de confidentialité du bot Discord

## 1. Introduction
Cette politique explique comment le bot collecte, utilise et protège les données des utilisateurs lorsqu’il est utilisé sur Discord.

## 2. Données collectées
Le bot peut collecter et stocker les informations suivantes :
- Identifiant Discord de l’utilisateur (`userId`)
- Données de profil liées au bot (par exemple collection de sprites)
- Données de transactions ou d’échanges entre utilisateurs
- Informations nécessaires au fonctionnement des commandes du bot

## 3. Utilisation des données
Les données sont utilisées uniquement pour :
- gérer les collections de sprites
- suivre les échanges et rapports
- fournir les fonctionnalités du bot
- améliorer le service et corriger les erreurs

## 4. Partage des données
- Les données ne sont pas vendues à des tiers.
- Les données ne sont pas partagées en dehors de ce qui est nécessaire pour faire fonctionner le bot.
- Les données peuvent être stockées sur la base de données du bot, uniquement pour le bon fonctionnement du service.

## 5. Stockage et sécurité
- Les informations sont stockées dans une base de données MySQL/MariaDB.
- Aucune donnée sensible (mot de passe, informations bancaires, etc.) n’est collectée par le bot.
- L’accès aux données est limité au bot et à son propriétaire / développeur.

## 6. Durée de conservation
- Les données sont conservées tant que l’utilisateur utilise le bot ou tant que le bot est actif.
- Si un compte utilisateur est supprimé ou si le bot est désactivé, les données peuvent être supprimées selon les besoins du propriétaire.

## 7. Droits des utilisateurs
- Vous pouvez demander la suppression de vos données liées au bot.
- Vous pouvez demander des informations sur les données stockées à votre sujet.
- Contactez l’administrateur du bot pour toute demande de ce type.

## 8. Modifications de la politique
- Cette politique peut être mise à jour à tout moment.
- En utilisant le bot après une mise à jour, vous acceptez les nouvelles conditions.

## 9. Contact
Pour toute question sur la vie privée ou les données, contactez le propriétaire/Développeur du bot, Renard Bleu.
